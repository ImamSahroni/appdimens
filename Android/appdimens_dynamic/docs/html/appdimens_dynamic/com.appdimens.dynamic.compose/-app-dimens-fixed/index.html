class AppDimensFixed(initialBaseDp: Dp, ignoreMultiViewAdjustment: Boolean = false)

Classe para construir dimensões "fixas" que são ajustadas automaticamente com base no menor smallestScreenWidthDp do dispositivo e na proporção da tela.
