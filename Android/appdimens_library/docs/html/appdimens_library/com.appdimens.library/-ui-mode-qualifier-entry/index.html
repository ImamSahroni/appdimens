UiModeQualifierEntry

data class UiModeQualifierEntry(val uiModeType: UiModeType, val dpQualifierEntry: DpQualifierEntry)

Representa uma entrada de qualificador que combina um tipo de UI Mode (dispositivo) E um qualificador de tela (SW, H, W). Esta combinação tem a PRIORIDADE MÁXIMA.

uiModeType

O tipo de UI Mode (CAR, TELEVISION, NORMAL, etc.).

dpQualifierEntry

O qualificador de tela (DpQualifier, valor DP mínimo).
