Fixed Scaling (FX)

Dimensões que crescem de forma logarítmica. Ideal para elementos de UI como botões, paddings e bordas.

Dynamic Scaling (DY)

Dimensões que crescem proporcionalmente. Perfeito para containers, grids e elementos que precisam de escala linear.

Fluid Scaling (FL)

Dimensões que crescem suavemente entre valores mín/máx. Ótimo para tipografia e elementos que precisam de transições suaves.
